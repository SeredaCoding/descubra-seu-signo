# Descubra seu Signo

Projeto web desenvolvido em PHP para consulta de signo do zodíaco basedo na data de nascimento.

## Descrição

Sistema simples que permite ao usuário inserir sua data de nascimento e obter informações sobre seu signo do zodíaco, incluindo período e características de personalidade.

## Tecnologias Utilizadas

- PHP
- HTML/CSS (Bootstrap)
- XML (armazenamento de dados dos signos)

## Como Executar

1. Certifique-se de ter o PHP instalado
2. Utilize um servidor local (XAMPP, WAMP, etc.)
3. Coloque os arquivos no diretório do servidor
4. Acesse através do navegador

## Objetivo

Relatório apresentado à Universidade Pitágoras Unopar/Anhanguera, como requisito parcial para o aproveitamento da disciplina de Programação Web, do curso Bacharelado em Engenharia de Software.
